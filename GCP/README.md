
# gcloud

```
アカウントログイン
$ gcloud auth login

認証済みアカウント一覧
$ gcloud auth list

アカウント切り替え
$ gcloud config set account アカウント名

SDKのアップデート
$ gcloud components update
ちなみに、gsutilのアップデートもこちらで行う模様
```
